# POC for Apty

* Design into static site 
* Desktop view 
* Mobile view (<575px)

#### Installation
1. Clone the repo
2. Install dependencies `npm install` 
3. Start the local server `npm start`
4. Load http://localhost:3000/

#### Built With
* React
* Custom CSS
